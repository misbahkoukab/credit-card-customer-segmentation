# credit-card-customer-segmentation
Power BI dashboard project for customer spending pattern segmentation using clustering analysis.
# Credit Card Spending Pattern Segmentation

This project focuses on customer segmentation using spending behavior analysis and clustering techniques. The cleaned dataset was analyzed and visualized using Power BI to identify different customer groups and spending patterns.

---

## Project Objective

- Analyze customer spending behavior
- Identify customer segments
- Generate business insights
- Visualize patterns through interactive dashboards

---

## Tools & Technologies

- Power BI
- Python
- Pandas
- K-Means Clustering
- Data Visualization

---

## Dashboard Features

- Customer Cluster Distribution
- Purchase Analysis
- Cash Advance Trends
- Balance Comparison
- Interactive Filters & Slicers
- Customer Segment Insights

---

## Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Scaling
5. K-Means Clustering
6. Power BI Dashboard Development
7. Business Insight Generation

---

## Key Insights

- Certain customer groups showed high purchase frequency
- Some customers relied heavily on cash advances
- High-credit-limit customers demonstrated unique spending patterns
- Different clusters reflected distinct financial behaviors
